# 第21章 GPT-1：通过生成式预训练提升自然语言理解

## 19.1 背景介绍

> The ability to learn effectively from raw text is crucial to
> alleviating the dependence on supervised learning in natural language
> processing (NLP).

"从原始文本中有效学习的能力对于减轻自然语言处理（NLP）对监督学习的依赖至关重要"。GPT-1
论文开篇首句如是说。这句话道出了 GPT 系列的初衷------利用无监督数据实现
Zero-Shot 解决各类问题。

深度学习长期依赖大量标注数据。GPT-1
首次系统性提出：通过大规模无监督预训练，再在下游任务上微调，可以显著提升性能并减少标注依赖。

### 面临的挑战

1.  哪种优化目标最有利于学习可迁移的上下文表示？\
2.  如何将学习到的表示有效迁移到不同任务？

------------------------------------------------------------------------

## 19.2 基本框架和方法

训练分为两个阶段：

### 19.2.1 无监督预训练

给定无监督文本 Token：

U = {u1, ..., un}

语言模型目标：

max ∑ log P(u_i \| u\_{i-k}, ..., u\_{i-1})

模型使用 Transformer Decoder。相比 RNN：

-   更好建模长依赖
-   更强迁移能力
-   更稳定训练

------------------------------------------------------------------------

### 19.2.2 有监督微调

标注数据：

C = {(x1,...,xm), y}

流程：

1.  输入经过预训练模型得到隐藏表示 h_lm
2.  加入线性层 W_y
3.  预测标签 y

目标函数：

L = L_task + λ L_LM

加入辅助语言模型目标可以提升泛化能力和收敛速度。

------------------------------------------------------------------------

## 19.3 实验结果

### 模型结构

-   12层 Transformer Decoder\
-   隐藏维度 768\
-   12个注意力头\
-   FFN维度 3072

### 数据集

-   BooksCorpus（预训练）
-   SNLI, MNLI, QNLI, SciTail, RTE
-   RACE, Story Cloze
-   MRPC, QQP, STS-B
-   CoLA, SST-2

实验表明：

-   预训练 + 微调 显著优于 直接监督训练
-   Transformer 优于 LSTM
-   层数越多迁移效果越好
-   零样本能力随训练步数提升
